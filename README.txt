Custom PHP MVC Framework

Features MVC:
- MVC architecture
- Advanced routing engine with pretty URLs
- Classes with namespaces and composer psr-4 autoload
- Controller with action filter
- Views in pure html with twig template engine
- Models with resource optimized database-connection-logic
- Environment specific configuration file and complete error/exception handling

Features Registration:
- User Registration
- Login
- Remember Me
- Password Reset
- Account activation via E-mail
- User Profil Edit

UI Features: 
- Bootstrap 4
- Flashmassages
- 404 and 500 Page

Installation:
- Rename ConfigCommit.php to Config.php and fill in the credentials
